Construct a tree using the trees constructor.  Decide on your goal then call search function.  It will find all available paths and then print them to you.  Then it will pick the first shortest path then print it also.  If you want it to be able to print more than one shortest path, use vector<int> instead of string and append using it.  If you want to use string but still want to find more than one shortest path, divide string to its tokens and count the numbers.  If you leave it as it and still try to find all shortest paths, it will fail because 2 digit numbers make the string longer.
